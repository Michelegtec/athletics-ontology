# athletics-ontology
Ontologia de Atletismo criada no WebProtégé, abrangendo atletas, eventos, classificações e recordes, baseada em literatura esportiva, DBpedia e Wikidata, seguindo padrões de dados conectados. Inclui questões de competência sobre definição de atleta, modalidades, classificações, recordes e relações entre atletas e eventos.
